# Piwik/Cache

This is a PHP caching library based on [Doctrine cache](https://github.com/doctrine/cache).

[![Build Status](https://travis-ci.org/piwik/component-cache.svg?branch=master)](https://travis-ci.org/piwik/component-cache)

## Installation

With Composer:

```json
{
    "require": {
        "piwik/cache": "*"
    }
}
```
